How to Use ColorInspo

Getting Started

When you first open ColorInspo, you'll see the main color generator screen. Here you can:

  • Tap anywhere on the screen to generate a new random color
  • Tap the color code to copy it to your clipboard
  • Use the format picker to switch between different color code formats
  • Tap the heart icon to favorite a color

Managing Colors

ColorInspo provides several ways to organize and manage your colors:

  • History: Access your recently generated colors in the history view
  • Favorites: Save colors you love by tapping the heart icon - they'll stay in your favorites until you remove them
  • Color Codes: Switch between HEX, RGB, HSL, HSB, and CMYK formats anytime

Color Variations

To explore variations of a color:

  • Select any color from your history or favorites
  • Tap "Show Variations" to see different shades, tints, and tones
  • Use the harmony options to find complementary, analogous, or triadic color combinations

Color Discovery BETA

The Color Discovery feature helps you find colors tailored to your taste:

  • Access the Color Discovery section from the main menu
  • Swipe right on colors you like, left on ones you don't
  • The app learns from your preferences to suggest better matches
  • View insights about your color preferences in the statistics section

Note: Color Discovery is a beta feature and may be refined or changed based on user feedback. The machine learning model runs entirely on your device and may take time to learn your preferences.

Tips & Tricks

  • Double-tap a color to quickly copy its code
  • Long-press a color in history to see additional options
  • Use the search bar in history to find specific colors by their code
  • Adjust your preferred color format in settings to see it by default

Troubleshooting

If you encounter any issues:

  • Ensure you're running the latest version of the app
  • Try force-closing and reopening the app
  • Check if you have sufficient storage space on your device
  • For beta features, try clearing the app's cache if you experience unexpected behavior

Modifying Colors

ColorInspo allows you to fine-tune any color using precise controls:

  • Tap the slider icon on any color to open the modify view
  • Switch between HSB (Hue, Saturation, Brightness) and RGB modes
  • Use precise sliders to adjust individual color components
  • See the original and modified colors side by side
  • Copy the modified color code in any supported format
  • Save your changes or reset to the original color
