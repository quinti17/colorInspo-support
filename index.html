How to Use

Generate Colors: Simply tap the "Generate" button to create a new random color.

Copy Color Code: Tap the displayed hex code to copy it to your clipboard.

View History: Tap the clock icon to see your previously generated colors.

Apply Previous Color: In the history view, tap any color to apply it to the main screen.
